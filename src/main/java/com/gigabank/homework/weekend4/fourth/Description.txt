### 4. Spring MVC: создание REST API

1. Создайте REST-контроллер `BankAccountController` и `TransactionController`.
2. Реализуйте следующие эндпоинты:
    - `POST /bankAccounts` — создать новый аккаунт
    - `GET /bankAccounts/{id}` — получить данные аккаунта
    - `POST /bankAccounts/{id}/deposit` — пополнить баланс
    - `POST /bankAccounts/{id}/withdraw` — снять средства
    - `POST /transactions/transfer` — перевести между счетами
    - `GET /bankAccounts/{id}/transactions` — получить историю операций
3. Используйте:
    - `@RequestMapping`, `@GetMapping`, `@PostMapping`
    - `@RequestParam`, `@RequestBody`, `@PathVariable`
