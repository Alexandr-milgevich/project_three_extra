## Подключить бд в проекте

В приложении создать класс `DBConnectionManager`для взаимодействия с базой данных.

Класс DBConnectionManagerдолжен иметь следующие элементы:

### **Поля класса**

- **Connection connection**: для хранения подключения к базе данных;
- **Properties props**: для хранения данных из `db.properties`, необходимых для подключения к БД.

### **Конструктор**

- Конструктор должен инициализировать поля и осуществлять подключение к базе данных.
- Реализация `DBConnectionManager` должна быть осуществлена через Singleton.

### **Методы**

- **connect()**: метод для открытия соединения с базой данных.
- **disconnect()**: метод для закрытия соединения.

Создать файл `db.properties` для хранения конфигураций подключения к бд.
