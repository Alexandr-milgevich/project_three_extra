## Создать таблицы

Для хранения данных в базе данных следует создать несколько таблиц. На данный момент планируется создание трех таблиц: `User`, `BankAccount` и `Transaction`.

### 1. Таблица `User` должна хранить информацию о пользователях.

- user_id - должен хранить идентификатор пользователя;
- username  - имя пользователя;
- email - электронная почта пользователя;
- phone - номер телефона пользователя.

```sql
  CREATE TABLE User (
       user_id VARCHAR(50) PRIMARY KEY,
       username VARCHAR(50) NOT NULL,
       email VARCHAR(100) NOT NULL UNIQUE,
       phone VARCHAR(15) UNIQUE
   );
```

### 2. Таблица `BankAccount` должна хранить информацию о счетах пользователей.

- user_id - должен хранить идентификатор пользователя;
- number - номер счета пользователя;
- balance - должен хранить балас счета пользователя.

### 3. Таблица `Transaction` должна хранить транзакции всех пользователей.

- id - идентификатор транзакции;
- user_id - должен хранить идентификатор пользователя.;
- amount - сумма транзакции;
- type - тип транзакции;
- date - дата транзакции;
- source - идентификатор пользователя от кого происходила транзакция. Если было пополнение или источник транзакции не был определен, оставить поле нулевым.
- target -  идентификатор пользователя кому проведена транзакция.
