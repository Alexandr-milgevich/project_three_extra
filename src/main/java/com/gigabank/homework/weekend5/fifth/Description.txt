## Копия базы данных для преподавателя

После создания базы данных, можно экспортировать из неё данные.
Например, с помощью инструмента mysqldump или аналогичного инструмента для вашей системы управления
базами данных (Если это потребуется ментору). Необходимо также предоставить данные для подключения к базе данных,
такие как URL, имя пользователя и пароль.
