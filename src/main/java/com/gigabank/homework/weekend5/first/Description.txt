## Создать базу данных в PostgreSQL

Первым шагом является создание базы данных, которая будет хранить всю необходимую информацию для приложения.
Это может быть произведено с помощью SQL-команд или  использованием pgAdmin.
Название базы данных должно быть выбрано в соответствии с названием приложения или его функционалом.
Важно учитывать подходящие настройки, такие как кодировка, и возможность расширения.
